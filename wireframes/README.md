# Wireframes — Kartka

Низькодеталізовані каркаси екранів та потоків. Мета — структура й потік, **не**
візуал. Чорно-біле, без кольору й фінальної типографіки.

## Що тут має з'явитися

Ключові потоки MVP (mobile-first):

- [ ] Онбординг + створення родини / першого профілю
- [ ] Головний екран (огляд родини)
- [ ] Профіль члена сім'ї + базова картка (алергії, хроніка, група крові)
- [ ] Додавання документа (фото/файл + дата, лікар, тип) — **швидкий шлях**
- [ ] Трекінг ліків (додати, історія, хто що приймає)
- [ ] Трекінг аналізів (додати результат, історія за параметром)
- [ ] Нагадування про візити (зокрема повторювані)
- [ ] Експорт анамнезу в PDF
- [ ] Екстрений екран

## Правила

- Mobile-first; десктоп-варіант — лише там, де поведінка відрізняється.
- На кожному екрані введення явно рахувати **кількість тапів/полів** (принцип №1
  у [`../CLAUDE.md`](../CLAUDE.md)).
- Іменування: `NN-flow-step.png` або `.excalidraw` / `.fig`-експорт.
