# Concept — Kartka

Високодеталізовані концепт-екрани: каркаси з вайрфреймів, одягнені у візуальний
напрям (теплий, сімейний) і токени. Це те, що показуємо як «ось так виглядатиме
продукт».

## Що тут має з'явитися

- [ ] Ключові екрани MVP у фінальному візуалі (з [`wireframes/`](../wireframes/))
- [ ] Світла тема (за потреби — темна)
- [ ] Стани: порожній / заповнений / помилка / завантаження
- [ ] Мобільний + десктоп-адаптив для 2–3 опорних екранів

## Залежності

- Візуальний тон і кольори — теплий, сімейний (див. [`../CLAUDE.md`](../CLAUDE.md), розд. 7).
- Кольори/типографіка/відступи беруться з [`../tokens/`](../tokens/).
- Компоненти — з [`../components/`](../components/) та [`../design-system/`](../design-system/).
