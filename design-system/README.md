# Design System — Kartka

Звід правил, що поєднує токени й компоненти в цілісну систему: принципи,
патерни, гайдлайни використання. Це «конституція» візуальної мови продукту.

## Зміст (план)

- **Принципи** — теплий сімейний тон; швидке введення понад усе; не медичний
  продукт (жодних оцінок/норм). Див. [`../CLAUDE.md`](../CLAUDE.md).
- **Фундамент** — посилання на [`../tokens/`](../tokens/): колір, типографіка,
  сітка, відступи.
- **Компоненти** — індекс із [`../components/`](../components/) з гайдами
  використання (коли який, що не робити).
- **Патерни** — композиції: форма швидкого додавання, список записів, таймлайн,
  порожні стани, екстрений екран.
- **Контент і мова** — людяний тон, українська + англійська (i18n), як писати
  про здоров'я без оцінок/порад.
- **Доступність** — контраст, розміри тапів, фокус, читабельність.

## Зв'язок

`tokens` → `components` → **`design-system`** → застосовується в `concept` і коді.
