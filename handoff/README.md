# Handoff — Kartka

Усе, що потрібно інженеру (тобто нам у фазі коду), щоб реалізувати дизайн без
здогадок: специфікації, експорти, мапа екранів, edge-кейси, готові ассети.

## Що тут має з'явитися

- [ ] Мапа екранів і навігації (sitemap / flow).
- [ ] Специфікації екранів: відступи, токени, поведінка, стани.
- [ ] Edge-кейси й порожні стани кожного екрана.
- [ ] Контент-стрічки / тексти (українська + англійська).
- [ ] Експорт ассетів (іконки, ілюстрації) + іменування.
- [ ] PWA-нотатки: manifest, іконки home-screen, поведінка офлайн.
- [ ] Нотатки по нагадуваннях: web push + запасний варіант (локальні / email),
      обмеження iOS.
- [ ] Контракт PDF-анамнезу: які поля, у якому порядку, верстка.

## Правила

- Handoff відображає **поточний** концепт; якщо концепт змінився — оновити тут.
- Нічого, що суперечить межам продукту в [`../CLAUDE.md`](../CLAUDE.md)
  (без AI, без оцінок/норм, доступ лише через RLS).
