# Design Tokens — Kartka

Атомарні значення дизайну: кольори, типографіка, відступи, радіуси, тіні,
тривалості анімацій. Єдине джерело правди для візуалу — звідси їх споживають
[`../components/`](../components/) і код.

## Формат

- `tokens.json` — машиночитані токени (W3C Design Tokens-сумісно, щоб легко
  експортувати в CSS-змінні / Tailwind-тему).
- Згенеровані `tokens.css` (CSS custom properties) — для прямого використання.

## Групи токенів (план)

- **color** — теплий сімейний палітр: базові, поверхні, текст, акцент, статуси
  (success / warning / danger), нейтральні.
- **typography** — шкала розмірів, ваги, line-height; українська + латиниця.
- **space** — крок відступів (4/8-базис).
- **radius** — м'які форми (відповідно до тону).
- **shadow** — легкі, неклінічні.
- **motion** — тривалості й easing.

## Правила

- Жодних «магічних» значень у компонентах — лише токени.
- Назви семантичні, не за виглядом (`color-text-muted`, не `color-grey-500`).
